# Spam Email Detection System

This is a machine learning application designed to detect and filter out spam emails. It utilizes a backend database to log and categorize incoming messages and features a graphical user interface for easy user interaction.

## Features
* **Machine Learning Classification:** Automatically detects and categorizes spam emails using dataset analysis.
* **Database Management:** Logs and stores processed message records securely.
* **User Interface:** Features an intuitive sidebar layout and an easy-to-use interface built for smooth project functionality.

## Technical Stack
* **Language:** Python
* **Libraries:** pandas, numpy, and Tkinter
